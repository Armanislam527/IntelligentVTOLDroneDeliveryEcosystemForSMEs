Smart Tension Sensor Mechanism

Automated safety feature for payload release

Automated String Release System

Our drones are equipped with a smart tension sensor that continuously monitors the load on the delivery string. If the weight exceeds the safe threshold (e.g., 5kg), the system automatically releases the string to prevent damage to the drone.
